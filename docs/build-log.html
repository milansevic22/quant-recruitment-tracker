Key Files

  • components/dashboard.tsx - dashboard UI and Firestore fallback loading.
  • lib/mock-data.ts - tracked firms, sample jobs, scan history, and review scan jobs.
  • lib/firebase/client.ts - browser-safe Firebase setup.
  • lib/firebase/admin.ts - server-only Admin SDK setup.
  • lib/scanner/scanner.ts - safe public careers-page scanning.
  • app/api/seed/route.ts and app/api/scan/route.ts - protected admin routes.
  • app/api/jobs/[jobId]/status/route.ts - protected persisted status updates.
  • app/api/cron/scan/route.ts and vercel.json - scheduled scan support.
  • firestore.rules - read-only client rules for dashboard collections.
