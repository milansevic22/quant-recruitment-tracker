CrumbLabz Internal MVP

Quant Recruitment Tracker

A Firebase-first internal dashboard for monitoring public careers pages and surfacing graduate, internship, quant, trading, research, and software engineering roles.

Project Goal

Build a polished internal MVP that turns a personal quant recruitment workflow into a reusable product surface. The app should be easy to review locally, ready for Firebase setup, and deployable on Vercel without a custom backend.

10 tracked firms in the initial seed set
3 Firestore collections for the core MVP
2 protected admin API routes
1 dashboard optimized for quick review

What This MVP Demonstrates

  • Planning-first delivery with clear implementation stages.
  • Next.js App Router, TypeScript, Tailwind CSS, and Firebase.
  • Safe public careers-page monitoring without bypass behavior.
  • Dashboard states that remain reviewable without credentials.
  • Deployment notes and a candid retrospective.

Intentional Simplicity

  • No custom Express backend.
  • No authentication until the product needs user accounts.
  • No PostgreSQL for this Firebase-first onboarding build.
  • No aggressive scraping or bot-protection workarounds.

Primary Review Path

  1. Read the product brief and architecture notes.
  2. Run the dashboard locally and confirm fallback data renders.
  3. Add Firebase credentials and seed the database.
  4. Run a manual scan through the protected API route.
  5. Deploy to Vercel after environment variables are configured.
